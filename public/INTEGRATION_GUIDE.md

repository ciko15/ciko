# Integration Guide for Airport/Cabang Monitoring

## Files Created:
1. `public/cabang-styles.css` - CSS styles for the monitoring interface
2. `public/cabang-app.js` - JavaScript functionality
3. `public/cabang-html.txt` - HTML section template

---

## Integration Steps:

### Step 1: Add CSS Link
Add this line in the `<head>` section of `public/index.html`, after the existing style.css link:

```html
<link rel="stylesheet" href="cabang-styles.css">
```

### Step 2: Add JavaScript
Add this line before the closing `</body>` tag, after app.js:

```html
<script src="cabang-app.js"></script>
```

### Step 3: Add Navigation Menu Item
In the sidebar navigation (`<nav class="sidebar-nav">`), add this new menu item:

```html
<a href="#" class="nav-item" data-section="cabang">
  <i class="fas fa-building"></i>
  <span>Cabang</span>
</a>
```

### Step 4: Add Monitoring Section HTML
Add this HTML inside `<main class="main-content">`, after the existing sections (before closing `</main>`):

Copy the content from `cabang-html.txt` and paste it in the index.html.

### Step 5: Add Sidebar Branch Panel
In the sidebar, add this panel after the navigation:

```html
<!-- Branch Search -->
<div class="sidebar-search">
  <i class="fas fa-search search-icon"></i>
  <input type="text" id="branchSearchInput" class="sidebar-search-input" placeholder="Search branch...">
</div>

<!-- Branch List -->
<div class="branch-list" id="branchListContainer">
  <!-- Branch items will be rendered here by JavaScript -->
</div>

<!-- Selected Branch Panel -->
<div class="branch-panel">
  <div class="branch-panel-header">
    <h4><i class="fas fa-check-circle"></i> Selected Branch</h4>
  </div>
  <div class="selected-branch-display" id="selectedBranchDisplay">
    <div class="branch-icon">
      <i class="fas fa-plane"></i>
    </div>
    <div class="branch-info">
      <div class="branch-name">Select Branch</div>
      <div class="branch-status">-</div>
    </div>
  </div>
</div>
```

## Features Included:

### Sidebar:
- Logo + Title: "Monitoring AirNav Indonesia" (can be customized)
- Search branch functionality
- List of airports/cabang with:
  - Status indicator (green/yellow/red/gray)
  - Airport name and city
  - Equipment count badge
- Selected Branch panel at bottom

### Main Content Area:
- Page title: "TOC – Remote Status Facilities"
- Category filters: All, Communication, Navigation, Surveillance, Data Processing, Support
- Stats summary bar showing equipment counts per category
- Equipment cards grid with:
  - Device name and code
  - Category icon
  - Status (Normal/Warning/Alert/Disconnect) with glow effect
  - Last update time
  - Signal indicator

### Technical Features:
- Dark modern theme (navy/dark blue)
- Responsive design (mobile/tablet/desktop)
- Modern hover effects
- Smooth animations
- Status colors with glow effects
- No external frameworks required (pure CSS)

## Customization:

### To change the title:
In `cabang-html.txt`, find:
```html
<h2>TOC – Remote Status Facilities</h2>
```

### To add more categories:
1. Add button in the filters div
2. Add stats-summary-item in the stats bar
3. Update the category mapping in `cabang-app.js`

### Colors:
The theme uses CSS variables from the main `style.css`. Main colors:
- `--accent-primary`: #2563eb (electric blue)
- `--accent-success`: #10b981 (green for Normal)
- `--accent-warning`: #f59e0b (yellow for Warning)
- `--accent-danger`: #ef4444 (red for Alert)
- `--accent-gray`: #6b7280 (gray for Disconnect)
